functions in C++
a function in C++ is a group of statements that performs a specific task 
and can be reused anywhere in tne program
	1. reusability - write once use many times
	2. readability - code becomee easier to read and understand
	3. debugging - errors are easier to find since each function is small and focused
	4. modularity - large programs are divided into smaller parts
	5. collaboration - multiply people can work on different functions of same program
